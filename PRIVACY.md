# 隐私政策

## 数据收集

本插件收集以下数据：
- 小红书笔记的标题、内容、图片URL
- 笔记的点赞数、收藏数、评论数
- 笔记的发布时间和作者信息

## 数据存储

所有数据**完全存储在您的本地浏览器中**：
- 使用 IndexedDB 本地数据库存储
- 使用 chrome.storage.sync 存储配置信息
- **不会上传到任何服务器**
- **不会与第三方共享**

## API 使用

当您启用 AI 分析功能时：
- 插件会将笔记内容发送到您配置的 AI API 服务商
- 发送的数据包括：笔记文本和图片URL
- 这些数据仅用于 AI 分析，不会被插件存储或共享
- 请查看您选择的 API 服务商的隐私政策
